What is a palindrome?

According to YourDictionary.com A palindrome is a word, phrase, number, or sequence of words that reads the same backward as forward. Punctuation and spaces between the words or lettering are allowed. Examples of palindromes include madam, racecar, redivider, etc. Read More

What is an anagram?

An anagram is a word or phrase that uses exactly the same letters to create a new word or phrase. For instance, "reed" is an anagram of "deer", and "anagram" is an anagram of "nag a ram".

About Palingram
As can be seen from the name, Palingram is a validator for palindromes and anagrams. Use Palingram to check if any given word or phrase is an palindrome, or if two words are anagrams of each other.
